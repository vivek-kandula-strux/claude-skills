Claude Skills — Web Development
A collection of Claude skill files for web development workflows.
responsive-web
Makes any web application responsive across mobile, tablet, laptop, and desktop without redesigning it, while improving load performance.
What it covers:

Box-sizing foundation and viewport meta
Fluid typography with clamp() — no typography media queries needed
Flexbox and grid collapsing at breakpoints
Hamburger nav with aria accessibility
44px touch targets for mobile usability
GPU-composited animations (no layout reflow)
Image lazy loading with preload for hero
Font loading via preconnect (no @import blocking)
Script defer/async strategy
dvw/dvh for modals (accounts for mobile browser chrome)
prefers-reduced-motion support

How to install
Download the .skill file and install it via Claude Settings → Skills.
